# KYC Data Quality & Compliance Analytics

A data analytics project focused on **validating KYC (Know Your Customer) records** to improve **data quality, accuracy, and regulatory compliance**. The project identifies duplicates, missing values, and anomalies using **SQL, Python, and Excel-style validation logic**.



## Project Overview

Financial institutions rely on accurate KYC data for regulatory compliance. This project analyzes **5,000+ customer KYC records** to:
- Detect duplicate customer records
- Identify missing or invalid values
- Validate data consistency and completeness
- Generate compliance-ready insights



## Objectives

- Validate customer KYC records
- Identify data quality issues
- Improve accuracy through structured checks
- Support compliance and audit requirements



## Tools & Technologies

- **SQL** – data validation & anomaly detection
- **Python (Pandas, NumPy)** – data cleaning & analysis
- **Excel** – reporting & summaries


---

## Key Data Quality Checks

- Duplicate customer detection
- Missing mandatory fields (PAN, Aadhaar, DOB)
- Invalid age and date formats
- Email and phone number validation
- Country and status consistency checks
